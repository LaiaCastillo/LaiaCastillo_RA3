# LaiaCastillo_RA3
# Resumen
El juego se llama Eres Un Bicho Bola El Juego, va de que efectivamente eres un bichobola y exploras el mundo de los bichos.
# Funcionalidades
- Movimiento del personaje + cambio de gravedad (AWSD+flechas)
- Sistema de vida + Muerte del player
- NPC con sistema de dialogos
- checkpoints
- inventario (reducido pq no me daba tiempo)
- Pinchos que hieren al player
- 4 pantallas tal y como pide la práctica (bastante sencillas)
- Cambio de escenas
- Menu de pause con boton de reiniciar y salir del juego implementado con eventos
- Menú de muerte
- Menú de Ganar
- Música de fondo
- Pool de flechas
- Enemigo
- Sonidos varios
- Items (Pociones)

## Sonidos
- El gusano cuando entras en su hitbox
- Múca de fondo
- Al tocar un checkpoint
- al salir el dialogo

## Controles
AWSD o flechas-> movimiento player
Tab -> Abrir inventario
Esq -> Pausar
E -> Usar Objeto
Espacio -> Cambiar gravedad
Click o Enter-> Pasar dialogos
